# TOOLS.md — Tartrinator

## Commandes utiles

- `openclaw status` — état du gateway
- `openclaw doctor` — diagnostic

## Sous-agents

Utiliser `sessions_spawn(...)` pour déléguer à des sous-agents :
- **Juridique** : conformité, rédaction documents légaux, RGPD
- **Technique** : développement feature, déploiement, tests
- **Stratégie B2B** : analyse marché, recommandations acquisition

## Tokens

Ne jamais exposer de tokens ou credentials. Utiliser les variables d'environnement ou le store de secrets.
